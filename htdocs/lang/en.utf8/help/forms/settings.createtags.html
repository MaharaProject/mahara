Create page via tags

You can select some tags to populate the page with content containing those tags.

The resulting page will contain the content (artefacts and blocks) that you tagged. You can then re-arrange the blocks on the page.

For example, if you tag:

  • One image and then select that tag here, it will be added as an image block to the page.
  • Three images with the same two tags and select these tags here, the images will be displayed in an image gallery.
  • One PDF and select that tag here, the PDF block will be added to the page.
  • An existing block on another page, e.g. the "Text" block, it will copy that block to this page.

Content mappings:

Type Count Block
Image 1 Image
2+ Image gallery slideshow
PDF 1 PDF block
2+ Files to download
Generic files 1+ Files to download
Audio/video 1 Internal media
2+ Files to download
Folder 1+ Folder block per folder
Journal 1+ Journal block per journal
Journal entry 1+ Tagged journal entries
Notes 1+ Note block per note
Plan/tasks 1+ Plan, i.e. the entire plan(s) that contain the tagged tasks

Note: The selected tags are only used to add the initial blocks to the page. If you tag other content afterwards, you would need to add that to the page yourself. Also, when you remove a block, it will not be added back automatically.

If you select a different tag (or the same again) the next time, the new content will be added to the page.
