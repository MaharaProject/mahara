Platform vendor

Technical knowledge of your LTI Advantage platform vendor required.

Consult your LTI Advantage documentation from your Platform.

The platform vendor points to the key in an LTI Advantage request that contains the session data from that vendor. This is used to identify the user making a request.

This is a unique key that will be related to the Platform vendor and looks like a URL.

An example of this key is presented in the LTI Advantage documentation here. In this example the key would be "http://www.ExamplePlatformVendor.com/session".

If your platform vendor is not in the list select 'Other' and enter the value provided by your platform.
