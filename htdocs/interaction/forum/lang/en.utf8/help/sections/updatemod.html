Update

Select topics by checking the box to the left of each topic. Then choose an action to update the selected topics.

Subscribe: You will receive notifications for posts to selected topics. This option is only available if you are not already subscribed to the entire forum.

Unsubscribe: You will not receive notifications for posts to selected topics. This is only possible when you are not subscribed to the forum.

Sticky: Selected topics will appear at the top of every page.

Unsticky: Selected topics will no longer appear at the top of every page.

Open: All group members will be able to post to selected topics.

Close: Only the group administrators and forum moderators will be able to post to selected topics.

Move to: Only the group administrators and forum moderators will be able to move selected topics to another forum if another forum exists.
